1. Клонируем проект git clone https://github.com/gmpero/diligent_js_meeting.git
2. В терминале, в корнивой папке проекта устанавливаем зависимости npm install
3. Установка браузеров Playwright npx playwright install
4. Запускаем базовые тесты и радуемся жизни npx playwright test --ui

5. Автотесты POM пишутся в папке tests.
6. Процедурные тесты(обычные) пишутся в папке proceduralTests

Алгоритм написания:
- Придумываете тест (По хорошему нам где-то вести описанные кейсы, хотя бы в виде чек-листа)
- Создаете отдельную ветку инициалы-название-проверки
- После разработки пушите изменения и делаете mergeRequest
- Ждете подтверждения и сливаете в main.

*Запрещено изменять конфигурационные файлы.* Можете менять их локально и внести в gitignore.
